Simple To-Do List App

Beginner

A classic project. Create a list that allows users to add, delete, and mark tasks as complete using local storage.

Tech Focus:

  • HTML, CSS, Vanilla JavaScript (DOM manipulation)
  • Local Storage API

Weather Dashboard

Intermediate

Fetch weather data from a public API (like OpenWeatherMap) and display current and forecasted conditions with custom icons.

Tech Focus:

  • Async/Await & Fetch API
  • Data processing (JSON)
  • Responsive design

Real-time Chat Application

Advanced

Build a multi-user chat interface using WebSockets for instant messaging, featuring user login and persistent messages.

Tech Focus:

  • Node.js (or Python) Backend
  • WebSockets (e.g., Socket.io)
  • Basic database (e.g., SQLite)
